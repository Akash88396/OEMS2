Formatting Rules

Your file must be saved as a .CSV (Comma Separated Values) file. You can do this in Excel by clicking "Save As" and choosing CSV.

Required Columns (In Order):

  1. Question Statement
  2. Option A
  3. Option B
  4. Option C (Leave blank if none)
  5. Option D (Leave blank if none)
  6. Correct Answer (A, B, C, or D)
  7. Difficulty (easy, medium, hard)

Note: The system will automatically skip the first row (Header row). Make sure your actual questions start on Row 2.
